Подключение скриптов

Подключение с помощью тега <script>

  • Встроенный скрипт (embedded)
  • Внешний скрипт (external)

Встроенный

index.html 
				<script>
					alert('Hello, world!');
				</script>

Внешний

index.html 
				<script src="script.js"></script>
script.js 
				alert('Hello, world!');

Размещение тега script

Интуиция подсказывает, что нужно размещать в теге head

			<head>
				<link rel="stylesheet" href="styles.css">
				<script src="script.js"></script>
			</head>
			<body>
				<!-- Rendering content -->
			</body>

Размещение тега script

Хорошая практика - подключать script в конце тега body

			<head>
				<link rel="stylesheet" href="styles.css">
			</head>
			<body>
				<!-- Rendering content -->
				<script src="script.js"></script>
			</body>

Структура кода

JavaScript код - набор команд, которые должен выполнить браузер.

Такие команды называются инструкциями (statements).

				statement;
				statement;
				statement;

Составная инструкция (block statement)

Позволяет объеденять несколько инструкций в одну.

				{
					statement;
					statement;
					statement;
				}

Переменные

Переменная - именованная ячейка памяти.

Объявляется с помощью ключевого слова var и идентификатора.

				var myVar;

Инициализация

Плохо

				var myVar;
				myVar = 'my var';

Хорошо

				var myVar = 'my var';

Единственная инструкция var

Плохо

				var name = 'Артем';
				var age = 24;

Хорошо

				var name = 'Артем',
					age = 24;

Область видимости

Область видимости переменной - функция в которой она объявлена.

			function foo(){
				var bar = 1;
				// do smth with bar
			}
			foo();
				
			console.log(bar); Uncaught ReferenceError: bar is not defined

Нету блочной области видимости

			{
				var foo = 'bar';
			}
			 
			console.log(foo); // bar

Динамическая типизация

Тип переменной определяется в момент присваивания значения

				var myVar = 1;
				myVar = "foo";
				myVar = true;

Оператор typeof

Определяет тип переменной

			typeof 1; // "number"
			typeof "some string"; // "string"
			typeof true; // "boolean"
			typeof undefined; // "undefined"
			typeof function(){}; // "function"

Для всего остального typeof возвращает "object", в том числе и для null.

undefined

  • Значение по умолчанию для переменных и параметров функции
  • Значение отсутствующих свойств объекта
			var myVar,
				o = {foo: 1};
			 
			typeof myVar; // undefined
			typeof o.bar; // undefined

null

Значение переменных которые ничего не содержат.

Плохо

			var activeItem = {};
			activeItem = undefined;

Хорошо

			var activeItem = {};
			activeItem = null;

Числа

  • Один тип для целых и дробных чисел
  • Хранятся в 64-битном формате
				1 // 1
				1.12 // 1.12
				0xff // 255
				1e3 // 1000
				1e-3 // 0.001

Потеря точности

			0.1 + 0.2 // 0.30000000000000004
			9999999999999999 // 10000000000000000

http://floating-point-gui.de/

NaN

Специальное значение: Not a Number

			"a" * 3 // NaN
			NaN == NaN // false
			isNaN(NaN) // true
			typeof NaN // "number"

Функция Number

Конвертирует значение в число

			Number("10") // 10
			Number("10.11") // 10.11
			Number("10a") // NaN
			+"10" // 10

Функция parseInt

			parseInt("10a", 10) // 10
			parseInt("a10", 10) // NaN
			parseInt("010") // в некоторых браузерах 8
			parseInt("010", 10) // во всех 10

Строки

  • Последовательность 16-битных символов
  • Двойные или одинарные кавычки
  • Нету char

Длина строки

			var myString = "my string";
			console.log(myString.length); // 9

Строки как массивы

			var myString = "my string";
			console.log(myString[0]); // "m"

Строки неизменяемы

			var myString = "my string";
			myString[6] = "o";
			console.log(myString); // "my string"

Операторы

  • Унарные +"1"
  • Бинарные 1 + 2
  • Тернарный ?:

Группы операторов

Арифметические
+ - * / %
Сравнения
< > <= >= == != === !==
Логические
! && ||
Присваивания
= *= /= %= += -=

+

			Если хотя бы один операнд строка
				конкатенация
			Иначе
				сложение
		
			'абв' + 'где' // 'абвгде'
			'' + 1 // '1' - компактнее String(1)

Сравнение с приведением типов

			'1' == 1 // true
			true == 1 // true
			true == 'true' // true

Сравнение без приведения типов

			'1' === 1 // false
			true === 1 // false
			true === 'true' // false
			1 === 1 // true

&&

			Если значение первого операнда истинное
				результат - второй операнд
			Иначе
				результат - первый операнд
		
			value && console.log(value);

||

			Если значение первого операнда ложное
				результат - второй операнд
			Иначе
				результат - первый операнд
		
			function foo(bar){
				bar = bar || "bar";
			}
